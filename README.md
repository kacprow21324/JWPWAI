## 💡 Jak zacząć

1. **Sklonuj repozytorium**  
   ```bash
   git clone https://github.com/kacprow21324/JWPWAI.git
   ```
2. **Przejdź do katalogu**  
   ```bash
   cd JWPWAI
   ```
3. **Otwórz stronę główną projektu**  
   Po prostu otwórz plik `index.html` w przeglądarce:  
   ```bash
   # macOS / Linux
   open index.html
   # Windows PowerShell
   start index.html
   ```
